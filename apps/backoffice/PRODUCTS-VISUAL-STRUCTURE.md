## API Request Examples

```
1. Fetch All Products
   GET http://backend:4000/api/products
   Response: Product[]

2. Fetch with Filters
   GET http://backend:4000/api/products?category=rug&status=available
   Response: Product[]

3. Fetch Single Product
   GET http://backend:4000/api/products/uuid-here
   Response: Product

4. Create Product
   POST http://backend:4000/api/products
   Body: CreateProductDto
   Response: Product

5. Update Product
   PATCH http://backend:4000/api/products/uuid-here
   Body: UpdateProductDto
   Response: Product

6. Delete Product
   DELETE http://backend:4000/api/products/uuid-here
   Response: 204 No Content
```
